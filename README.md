### 项目介绍
本人在班级担任电子教学管理员的职位，受数学老师使用Deepseek制作的打卡程序启发，制作了本程序。

### 程序介绍
此版本为此项目上传的第一个离线版本，给班级大屏无法联网的班级使用。

你可以使用config.ini配置文件配置你的班级信息。

该配置文件包含如下信息:
```
  [config]
  class_id =      //班级
  nj=             //年级
  z=7             //排版横行
  l=7             //排版竖行
  km=             //科目
  school=         //学校
```

## 如何添加学生信息？
在name.txt中添加学生信息，格式如下：
```txt
1
2
3
```
